link_externals¶

link_externals(paths, file_list, mapping_dict={})¶

Create symlinks to externals from list of files containing linking instructions.

Parameters

  • paths (dict) –

    Dictionary of paths. Dictionary should contain {
    ‘external_dir’str

    Directory to write symlinks.

    ’makelog’str

    Path of makelog.

    }

  • file_list (list) – List of files containing linking instructions.

  • mapping_dict (dict, optional) – Dictionary of path mappings used to parse linking instructions. Defaults to no mappings.

Returns

move_map – List of (source, destination) for each symlink created.

Return type

list
